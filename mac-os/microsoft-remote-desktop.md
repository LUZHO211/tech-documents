# 在 Mac 上使用远程桌面

在个人`Mac OS`电脑上，可以使用`Microsoft Remote Desktop for Mac`软件来使用`Windows`的远程桌面。

下载地址：[Microsoft Remote Desktop for Mac](https://rink.hockeyapp.net/apps/5e0c144289a51fca2d3bfa39ce7f2b06/)
